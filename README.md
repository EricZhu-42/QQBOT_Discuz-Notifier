## 开发环境
开发过程中使用的服务器环境与第三方模块版本如下：
- **Python** = 3.7.1  
- **JRE** = java version **"11"**, Java(TM) SE Runtime Environment 18.9
- **mirai-console-wrapper** = 1.2.0 （封装后的mirai jar包）
- **mirai-http-api** = 1.7.0 （mirai插件）
- **kuriyama** = 0.3.2 （Python第三方库）
- **Requests** = 2.23.0
- **BeautifulSoup** = 3.2.2
